О нас

Межкомнатные двери Портас - отличный вариант для тех, кто ищет доступный вариант по цене из экошпона. Серия дверей Portas представлена полотнами с разным видом вставок стекла. Для остекления дверей используется Мателюкс, на котором не остаются отпечатки пальцев.

Основные направления
  • Царговая система сборки высокого качества (любая деталь подлежит замене).
  • Для покрытия используется немецкая плёнка ПВХ: износостойкая, хорошо защищает полотно от влаги.
Последовательность проведения работ:
  1. коробка квадратная с уплотнителем 32х74х2070 мм
  2. наличник прямой телескоп. 10х70х2150 мм
  3. добор 100 телескоп 10х100х2070 мм
  4. добор 150 телескоп 10х150х2070 мм
  5. притворная планка 10х30х2070 мм
  6. Коробка с уплотнителем + LVL-брус
